    Part 2 (Perspective Projection):
    Here we make a MVP for each cube.
    First I define the centering (C) as (-0.5, -0.5, -0.5) to center the cube at the origin. Then I can transform each cube by translating its postion and rotation.
  • MVP_1 = M_st x M_persp x V x T(-2, 0, 0) x C
  • MVP_2 = M_st x M_persp x V x T(0, 0, 0) x R_y(30) x C
  • MVP_3 = M_st x M_persp x V x T(2, 0, 0) x R_x(30) x R_y(30) x C

  • A more general form for MVP_i would be:
    MVP_i = M_st x M_projection x V x M_model
    As can be seen on line 92 (M_projection) and line 105.
    where M_model is the model matrix for each cube, which includes translation and rotation.
