  1. Flat shading: Each triangle/face is shaded based on the triangles normal vector. Resulting in a single color pr face
    Gourand shading: Instead of computing the color pr face, the color is computed pr vertex and then interpolated across the face. This results in a smoother appearance.
    Phong shading: The normal vector is interpolated across the face and the color is computed pr pixel.
    Implementing flat shading: Loop over each triangle, compute the face normal and use this normal for all pixels in the triangle when computing the color.
    Phong shading is better. The highlight are sharper and the overall appearance is smoother. Where Gourand shading can miss highlights if they are not located at a vertex.
  2. Directional light has the same intensity regardless of the distance from the light source to the object.
    Point light intensity decreases with distance from the light source to the object.

  4. Setting the specular term/coefficient to (0, 0, 0,) (k_s = 0) will remove the highlights. (Has been done in the above model)
  5. Increasing the shininess (s) creates smaller and sharper highlights.
